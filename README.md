# Hitesh Kaurav Portfolio

A professional static portfolio website for Hitesh Kaurav, built with Vite and React.

The copy uses conservative public profile information from Hitesh's LinkedIn profile:
<https://in.linkedin.com/in/hitesh-kaurav>

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
